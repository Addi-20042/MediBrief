## Standards for Every Public Repository

- Clear repository name
- One-line description
- Helpful README with stack, features, and setup
- At least one screenshot, architecture diagram, or preview asset when possible
- Meaningful commit messages
- No secrets, generated junk, or broken links

## Repo Cleanup Checklist

- Add a one-line repo description on GitHub
